# Agenthoryx Certification Program

The Agenthoryx Foundation offers an official certification path to validate the skills of developers working with AI-native systems.

## Certification Tracks

### 1. Agenthoryx Associate Developer (CAD)
- **Target Audience**: Junior developers and students.
- **Topics**: Basic syntax, compilation pipeline, using `ai.chat()` and `ai.embed()`, understanding basic agent creation.
- **Prerequisites**: None.

### 2. Agenthoryx Professional Developer (CPD)
- **Target Audience**: Mid-level developers.
- **Topics**: Custom VM integration, writing native extensions, advanced memory management, managing package dependencies via CPM.
- **Prerequisites**: CAD.

### 3. Agenthoryx AI Engineer (CAE)
- **Target Audience**: ML/AI developers.
- **Topics**: Fine-tuning agent behavior, optimizing context windows, utilizing the Memory API for vector searches, multi-model routing.
- **Prerequisites**: CPD.

### 4. Agenthoryx Agent Architect (CAA)
- **Target Audience**: Senior system designers.
- **Topics**: Distributed agent execution, designing robust Workflow DAGs, handling non-deterministic failure states, securing multi-agent communications.
- **Prerequisites**: CAE.
