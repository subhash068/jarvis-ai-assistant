# Agenthoryx Academic Curriculum

The Agenthoryx Academic Initiative partners with universities globally to introduce AI-native programming into computer science programs.

## Undergraduate Tracks

### 1. Programming Fundamentals with Agenthoryx (CS101)
- Introduction to Syntax and Types
- Control Flow and Functions
- Introduction to Non-Deterministic Logic (Using `ai.chat()`)
- *Lab: Building a basic text adventure with an AI-driven NPC.*

### 2. Compiler Design (CS301)
- Lexical Analysis and Tokenization
- AST Generation
- Bytecode Emission and Stack Machines
- *Lab: Implementing a custom OpCode in the open-source Agenthoryx VM.*

### 3. AI Systems (CS401)
- LLM Integrations in Software Engineering
- Prompt Engineering vs Code Logic
- *Lab: Building a RAG (Retrieval-Augmented Generation) application using `ai.embed()`.*

## Postgraduate Tracks

### 1. Agent Systems Engineering (CS501)
- Stateful Agents and Context Management
- Orchestration and Workflows
- *Lab: Building a multi-agent debate simulation.*

### 2. Distributed AI Architecture (CS601)
- Distributed Run-times and Node Execution
- Securing Multi-Agent Systems
- *Lab: Deploying a 50-node agent cluster on Agenthoryx Cloud.*
