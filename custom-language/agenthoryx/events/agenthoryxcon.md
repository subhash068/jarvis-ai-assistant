# AgenthoryxCon

AgenthoryxCon is the premier annual international conference for developers, researchers, and enterprises building the future of AI-native software.

## Conference Tracks

### 1. Programming Languages & Compilers
Deep dives into the internal architecture of the Agenthoryx VM, garbage collection for agents, and AOT compilation strategies.

### 2. AI Systems & Runtime
Showcases of novel LLM integration patterns, optimized embeddings, and vector database benchmarks using the Agenthoryx Memory API.

### 3. Agent Engineering
Case studies on building reliable, deterministic workflows using non-deterministic autonomous agents. Topics include prompt robustness, state management, and debugging.

### 4. Enterprise & Cloud Computing
Discussions on securing multi-agent systems via RBAC, scaling Agenthoryx Cloud deployments, and managing large-scale enterprise workflows.

### 5. Developer Tooling
Presentations on the evolving Agenthoryx ecosystem, including Package Manager updates, VS Code extension enhancements, and new SDK releases.
