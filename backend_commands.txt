Next Steps to Run Locally
Since you have PostgreSQL natively installed, follow these steps in your terminal to initialize the database and run the server:

1. Create the Database (if not already created): Ensure you have created a database named jarvis_ai in your local PostgreSQL instance. You can do this via psql or pgAdmin.

2. Set up the Python Environment: Navigate to the backend folder, create a virtual environment, and install dependencies:

cd backend
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
3. Generate and Apply Database Migrations: We use Alembic to manage database schema changes. Generate your first migration and apply it:

alembic revision --autogenerate -m "Initial migration"
alembic upgrade head
4. Run the FastAPI Server:

fastapi dev main.py
Your API will be running at http://localhost:8000. You can visit http://localhost:8000/docs to see the automatically generated interactive API documentation (Swagger UI).
